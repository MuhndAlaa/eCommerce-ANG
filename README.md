# Angular E-Commerce App

A angular project for e-commerce website


## Run Locally

Install dependcies

```bash
  npm intsall
```

To build, run the following command

```bash
  npm serve --o
```

To start live server, run the following command

```bash
  npm build --prod
```

## Home endpoint

```bash
  http://localhost:4200/
```


## Tech Stack

**Client:** HTML, CSS , bootstrap, typescript, angular
